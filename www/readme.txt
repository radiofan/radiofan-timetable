index.php - основа сайта. Запускает движок.
Операции:
 Объявление главной константы MAIN_DIR (если она не объявлена значит к срипту попытались обратиться без использования index.php)
 Создание констант (defines.php)
 Объявление функций (functions.php)
 Подключение логов если USE_LOG истина (переменная $LOG) (смтр. includes/log-class.php)
 Подключeние к БД (переменная $DB) (смтр. includes/db-class.php)
 Чтение опций, настроек (переменная $DATA) (смтр. includes/data-class.php)
 Проверка сессии
 Объявление пользователя (переменная $USER) (смтр. includes/user-class.php)
 Объявление ЧПУ (переменная $URL) (смтр. includes/url-class.php)
 Создание страниц (смтр. pages.php)
 Выполнение событий (смтр. actions.php)
 Обработка параметров (смтр. pages.php) для шаблонизатора, генерация выбранной страницы, её показ.

defines.php - файл хранящий константы.
 DATA_IN_DB - 0(данные хранятся в файле), 1(данные хранятся в бд)
 CLEAR_POST - 0(после пост запроса данные не очищаются), 1(перезагружает страницу, вследствии чего данные пост запроса очищаются)
 SALT - [string] (соль пароля)
 также константы подключения к БД

functions.php - файл с функциями

includes/log-class.php - Файл хранящий описание класса (rad_log) для записи ошибок и исключений в файл files/debug.log

includes/db-class.php - Файл хранящий описание класса (rad_db) для работы с БД (смтр. http://phpfaq.ru/safemysql)

includes/data-class.php - файл хранящий описание класса для работы с параметрами

includes/user-class.php - Файл хранящий описание класса для работы с пользователем

includes/url-class.php - файл хранящий описание класса для работы со страницами и ЧПУ

header.php, footer.php - файлы выводящие подвал и заголовок странцы, подключаются при создании страницы

pages.php - содержит функции для задания страниц и обработки данных для шаблонизатора

actions.php - содержит функции обработки событий, действий (POST или GET запросы)

ajax.php - файл принимающий ajax запросы
